# GBD-COVID-Mental
This repository contains all the codes used for the GBD-COVID-Mental manuscript.       
       
## Code  
  
The **Code** folder contains all the codes used in this study.   
  
**Codes**        
- 20260430_excess_burden_ARIMA_ARIMA-LSTM_Combined.R- Estimates counterfactual mental health burden using ARIMA and ARIMA–LSTM models, compares validation performance, selects the better-performing model, and calculates excess burden during the pandemic period.       
- 20260430_excess_burden_ITS.R- Conducts interrupted time-series (ITS) sensitivity analyses to estimate pandemic-associated excess mental health burden using an alternative modelling approach.   
- 20260430_policy_indices.R- Constructs annual government policy response indices from OxCGRT indicators, including domain-level indices such as containment and closure, economic support, health-system measures, vaccination, border control, and facial covering.        
- 20260430_excessburden_policyindex.R- Links estimated excess mental health burden with policy-domain indices and performs the main policy-domain association analyses.           
- 20260430_excessburden_policyindicator.R- Examines associations between excess mental health burden and individual OxCGRT policy indicators.           
- 20260430_independentassociation_excessburden_policyindex.R- Estimates mutually adjusted associations between policy-domain indices and excess mental health burden, including IQR-scaled effects to improve interpretability.               
- 20260430_interaction.R- Performs interaction analyses to examine whether policy domains or national structural indicators modified the associations between policy responses and excess mental health burden.
